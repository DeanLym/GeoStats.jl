Sampling

Overview

Samples can be drawn from spatial objects:

StatsBase.sample — Method
sample(object, sampler)

Sample elements from object with sampler.

StatsBase.sample — Method
sample(object, nsamples, [weights], replace=false)

Generate nsamples samples from spatial object uniformly or using weights, with or without replacement depending on replace option.

Example

Z = [norm([i,j]) for i in 1:100, j in 1:100]
Ω = RegularGridData{Float64}(OrderedDict(:Z=>Z))

S = sample(Ω, 1000, replace=false)

plot(plot(Ω), plot(S))

Methods

GeoStatsBase.UniformSampler — Type
UniformSampler(size, replace=false)

A method for uniform sampling from spatial objects that produces samples of given size with or without replacement depending on the option replace.

GeoStatsBase.BallSampler — Type
BallSampler(radius; [options])

A method for sampling isolated points from spatial objects using a ball neighborhood of given radius.

Options

  • metric - Metric for the ball (default to Euclidean())
  • maxsize - Maximum size of the resulting sample (default to none)
GeoStatsBase.WeightedSampler — Type
WeightedSampler(size, [weights]; replace=false)

A method for weighted sampling from spatial objects that produces samples of given size based on weights with or without replacement depending on the option replace. By default weights are uniform.
