Weighting

Overview

Points or samples in spatial objects can be weighted:

GeoStatsBase.weight — Method
weight(object, weighter)

Weight spatial object with weighter method.

Methods

GeoStatsBase.BlockWeighter — Type
BlockWeighter(side)

A weighting method that assigns weights to points in spatial object based on blocks of given side. The number (n) of points inside a block determines the weights (1/n) of these points.

GeoStatsBase.DensityRatioWeighter — Type
DensityRatioWeighter(tdata; [options])

Density ratio weights based on empirical distribution of variables in target data tdata.

Optional parameters

  • variables - Variables to consider (default to all)
  • estimator - Density ratio estimator (default to LSIF())
  • optlib - Optimization library (default to default_optlib(estimator))

Notes

Estimators from DensityRatioEstimation.jl are supported.
