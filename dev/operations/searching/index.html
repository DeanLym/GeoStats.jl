Searching

Overview

Neighbors of a point or sample can be searched efficiently:

GeoStatsBase.search — Method
search(xₒ, searcher, mask=nothing)

Return neighbors of coordinates xₒ using searcher and a mask over the spatial object.

GeoStatsBase.search — Method
search(location, searcher, mask=nothing)

Return neighbors of location in spatial object using searcher and a mask.

Example

Z = [norm([i,j]) for i in 1:100, j in 1:100]
Ω = RegularGridData{Float64}(OrderedDict(:Z=>Z))

# construct searcher
b = BallNeighborhood{2}(20.)
s = NeighborhoodSearcher(Ω, b)

# query neighbors of point
I = search([50., 50.], s)
N = view(Ω, I)

p₁ = plot(Ω)
p₂ = plot(N, lims=(0,100))

plot(p₁, p₂)

Methods

GeoStatsBase.NeighborhoodSearcher — Type
NeighborhoodSearcher(object, neighborhood)

A method for searching neighbors in spatial object inside neighborhood.

GeoStatsBase.NearestNeighborSearcher — Type
NearestNeighborSearcher(object, k; locations=all, metric=Euclidean())

A method for searching k nearest neighbors in spatial object locations according to metric.

GeoStatsBase.BoundedSearcher — Type
BoundedSearcher(searcher, nmax)

A method for searching at most nmax neighbors using searcher.
